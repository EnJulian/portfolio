# Portfolio (web-cv)

## Requirements

- Node.js `18.17.0` or newer
- npm

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```

The app vendors its non-Terminus fonts through local npm packages, so builds work offline after `npm install` has completed.
